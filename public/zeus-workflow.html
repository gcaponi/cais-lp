I 3 Motori Cognitivi
Il cuore di ZEUS — come ogni motore trasforma dati in conoscenza
Motore A

Gap Engine

Valuta la sufficienza delle risposte del cliente. Identifica lacune, genericità e contraddizioni. Genera domande di approfondimento mirate.

Input
  • Pre-DNA (6 layer bozza)
  • Risposte alle domande (tutti i round)
  • Piano cliente (profondità)
Processo
  • LLM valuta ogni risposta su 3 criteri
  • sufficiente / insufficiente / contradicts
  • Genera follow-up per risposte deboli
Output
  • Valutazioni per domanda
  • overall_sufficient: true/false
  • Follow-up mirati (F1, F2...)
PianoMax RoundMax Follow-up
Foundation13
Professional25
Legacy320
Motore B

Cross Specialist

Confronta i DNA Specialisti attivi tra loro e con il DNA Generale. Identifica pattern condivisi, conflitti e propone consolidamenti.

Input
  • DNA Generale (completo, corrente)
  • 2+ DNA Specialisti (status: attivo)
Processo
  • LLM analizza convergenze/divergenze
  • Identifica pattern trasversali
  • Genera proposte di consolidamento
Output
  • Shared patterns (temi comuni)
  • Conflicts (gravità + raccomandazione)
  • Proposte (layer + valore proposto)
Motore C

Coerenza Audit

Audit T3 periodico che confronta DNA Generale vs Specialisti. Trova contraddizioni, generalizzazioni indebite e regole mancanti.

Input
  • DNA Generale (completo)
  • Tutti gli Specialisti attivi (o uno specifico)
Processo
  • LLM confronta layer cognitivi vs tecnici
  • Max 12 issue per audit
  • Temperatura bassa (0.25, 0.15, 0.05)
Output
  • ConsistencyIssue records
  • contradiction / generalization / missing_rule
  • stale_general / boundary
